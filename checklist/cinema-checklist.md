# Чек-лист тестирования приложения «Онлайн-кинотеатр»

## Источники
1. Apple Human Interface Guidelines — https://developer.apple.com/design/human-interface-guidelines
2. App Store Review Guidelines — https://developer.apple.com/app-store/review/guidelines
3. ISTQB Mobile Application Testing + Ministry of Testing (ministryoftesting.com)

## 1. Установка и запуск
- [ ] Приложение устанавливается из App Store без ошибок
- [ ] Первый запуск: корректный сплэш-экран, нет крашей
- [ ] Запросы разрешений (уведомления, память) понятны и корректны
- [ ] Корректный запуск после обновления, данные сохранены
- [ ] Иконка и название отображаются правильно

## 2. Тёмная / светлая тема
- [ ] Поддержка обеих тем, реакция на системную настройку
- [ ] Все экраны перерисовываются без «слепых» зон
- [ ] Текст читаем на обоих фонах (контраст)
- [ ] Постеры, кнопки, иконки корректны в обеих темах
- [ ] Смена темы во время просмотра не прерывает воспроизведение

## 3. Работа с сетью (включая 3G)
- [ ] Контент грузится на 3G, есть индикатор загрузки
- [ ] Адаптивное качество видео при слабой сети
- [ ] Корректная обработка потери связи (сообщение, без краша)
- [ ] Возобновление после восстановления сети
- [ ] Переключение Wi-Fi ↔ мобильная сеть не прерывает просмотр

## 4. Push-уведомления
- [ ] Приходит push о выходе новой серии
- [ ] Тап по push открывает нужный экран (deep link)
- [ ] Нет уведомлений при отключённом разрешении
- [ ] Корректный текст и локализация
- [ ] Нет дублирующихся уведомлений

## 5. Возобновление просмотра
- [ ] Просмотр продолжается с точного места прерывания
- [ ] Прогресс сохраняется после перезапуска приложения
- [ ] Синхронизация прогресса между устройствами (если есть)
- [ ] Восстановление после входящего звонка/сворачивания

## 6. Офлайн-режим (скачивание)
- [ ] Фильм скачивается, виден прогресс
- [ ] Скачанное воспроизводится без интернета
- [ ] Сообщение при нехватке места в памяти
- [ ] Удаление скачанного освобождает память
- [ ] Корректная обработка истечения лицензии офлайн-контента

## 7. Сценарии прерывания
- [ ] Входящий звонок ставит на паузу и возобновляет
- [ ] Сворачивание/возврат сохраняет состояние плеера
- [ ] Блокировка экрана не сбрасывает прогресс
